# NHL Draft Value by Team (2006–2025)

An interactive visualization of which NHL teams drafted players who produced the
most career games, goals, assists, and points — broken down by draft round and
in total. Covers all 20 entry drafts from 2006 to 2025.

**[View the live report →](https://kevinohashi.github.io/nhl-draft-value/)**

Each pick is credited to the team that drafted it, using the team name **as it
was at the time** (so relocated franchises are kept separate). Stats are each
player's career NHL totals. Source data: [hockey-reference.com](https://www.hockey-reference.com/).

The page is a single self-contained `index.html` (data inlined as JSON; charts
rendered with [Chart.js](https://www.chartjs.org/) from a CDN).
